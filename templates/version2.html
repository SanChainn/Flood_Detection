ABOUT

This is Flood Detection System . We know Our Mon State is in the danger by Flooding . Many houses and families are damaged by these Disasters .So we decided to make this project protect from these Disasters .

In this project , We use IOT(Internet of Thing) technology which combines WEB Base and Hardware . So We develop front-end Web View and Back-end for processing and then We write Low-level program for control SENSOR , LED , BUZZER ,etc.
